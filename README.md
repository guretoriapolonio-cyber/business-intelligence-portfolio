# Business Intelligence Portfolio

Olá! Sou Gustavo Apolonio e este repositório reúne meus projetos de Business Intelligence e Data Analytics desenvolvidos em Power BI e SQL Server.

## Projetos

### 🎫 Service Desk Analytics — Gestão de Chamados de TI
Modelagem de banco de dados relacional para um sistema de Service Desk (usuários, técnicos, categorias, prioridades, status e chamados), com script SQL de criação e carga das tabelas. Estrutura preparada para dashboard de indicadores de SLA, tempo de resolução e distribuição por categoria/técnico.

**Arquivo:** `service_desk_schema.sql`

### 📊 Sales Performance Dashboard
Dashboard de análise de vendas com KPIs, evolução temporal e segmentação por filtros.

**Arquivo:** `PowerBI_Sales_Performance_Dashboard.pbix`

### 👥 HR Analytics Dashboard
Análise de colaboradores, turnover e indicadores de RH.

**Arquivo:** `HR_Analytics_Dashboard.pbix`

## Ferramentas Utilizadas

- Power BI
- DAX
- Power Query
- SQL Server
- Modelagem de Dados
